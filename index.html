Weather App

The Weather App provides real-time weather updates and a 5-day forecast for any city. Users can also view weather conditions based on their current location, with weather icons for better visualization. The app is built using Flask (Python) for the backend and HTML/CSS/JavaScript for the frontend, fetching weather data from the OpenWeatherMap API. It features an info button explaining the PM Accelerator. The app is easy to set up and run locally with minimal dependencies.

Receipt_Processor

The Receipt Processor is a Java-based Spring Boot application designed to process receipts and calculate reward points based on predefined rules. Users can submit receipts via a REST API, and the system assigns a unique receipt ID while computing points based on retailer name, total amount, item details, and purchase date/time. A second API allows users to retrieve the awarded points using the receipt ID. The application runs in-memory storage for quick processing and is Dockerized for easy deployment.
